# HTML/CSS YouTube Clone

This project is a static front-end clone of a YouTube homepage built with HTML and CSS. It recreates the look and layout of a typical video-sharing website, including a top header, left sidebar, and a grid of video cards.

## Overview

The goal of this project is to practice core HTML and CSS skills such as:

- Structuring a page with semantic HTML
- Styling layout with CSS
- Building a responsive-looking grid of content cards
- Using images, icons, and custom typography
- Creating a polished user interface inspired by YouTube

## Features

- YouTube-style header with logo, search bar, and action icons
- Sidebar navigation with icons and labels
- Video card layout with thumbnails, titles, channel names, and view counts
- Hover-style tooltips on some icons and buttons
- Clean, modern styling using external fonts and CSS files

## Project Structure

- youtube.html — Main HTML page for the project
- stylesyt/ — CSS files for the overall layout and components
  - general.css
  - header.css
  - video.css
  - sidebar.css
- icons/ — Header and UI icons
- sidebar_icons/ — Sidebar navigation icons
- thumbnails/ — Thumbnail images for video cards
- channel_pictures/ — Channel profile images

## How to View the Project

Open the file in a browser:

1. Navigate to the project folder
2. Open youtube.html in your preferred browser

You can also serve the folder locally with a simple HTTP server if you want:

```bash
python -m http.server 8000
```

Then open http://localhost:8000 in your browser.
